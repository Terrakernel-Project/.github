# Terrakernel Project

The official organization for the Terrakernel project - a 64-bit AMD64 operating system and kernel written from scratch in C/C++ and Assembly.

---

## About

Terrakernel is a hybrid x86_64 kernel being developed as the foundation for Terra, a modern operating system. The project focuses on building a complete OS from the ground up with clean implementations of core subsystems.

**Current Status:** Active development towards v1.0-rc1

**Statistics:**
- ~60,000 lines of code
- Primary language: C++
- Architecture: AMD64 (x86_64)
- License: BSD 3-Clause License

## Project Structure

**[Terrakernel](https://github.com/Terrakernel-Project/Terrakernel)** - The main kernel repository containing all kernel code, drivers, and subsystems. See the repository README for detailed information about features, building, and development.
